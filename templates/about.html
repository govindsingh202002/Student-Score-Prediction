About This Project

The Student Exam Performance Prediction system is a machine learning powered web application that estimates a student's Math Score based on academic and demographic inputs. The project demonstrates how predictive analytics can be used to support early guidance and academic improvement.

Purpose

The purpose of this project is to help teachers, mentors, and students identify potential academic challenges before exams occur, enabling early action toward improvement.

How It Works

  • User submits student-related information through the form.
  • Input data is preprocessed (encoding + scaling).
  • A trained regression model predicts the math score.
  • The predicted score is displayed on-screen.

Technology Stack

  • Backend: Python, Flask
  • Model:Machine learning regression models such as LinearRegression, XGBOOST, RandomForest
  • Frontend: HTML, CSS (Jinja2 templates)
